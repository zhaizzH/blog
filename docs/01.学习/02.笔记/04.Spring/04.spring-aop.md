# Spring AOP

## 概念

![AOP 概念](https://img.zhaizz.top/file/blog-website/spring/1630144353462(1).png)

| 概念 | 说明 |
|------|------|
| 连接点（JoinPoint） | 程序执行过程中的任意位置（方法执行） |
| 切入点（Pointcut） | 匹配连接点的表达式，需要增强的方法 |
| 通知（Advice） | 共性功能，最终体现为一个方法 |
| 通知类 | 定义通知的类 |
| 切面（Aspect） | 描述通知与切入点的对应关系 |
| 目标对象（Target） | 被代理的原始对象 |
| 代理（Proxy） | 对目标对象进行功能增强的对象 |

## 入门案例

```java
@Component
@Aspect
public class MyAdvice {
    @Pointcut("execution(void com.itheima.dao.BookDao.update())")
    private void pt(){}

    @Before("pt()")
    public void before(){
        System.out.println(System.currentTimeMillis());
    }
}
```

```java
@Configuration
@ComponentScan("com.itheima")
@EnableAspectJAutoProxy
public class SpringConfig { }
```

## 切入点表达式

![切入点表达式](https://img.zhaizz.top/file/blog-website/spring/1630155937718(1).png)

```
execution(访问修饰符  返回值  包名.类/接口名.方法名(参数) 异常名)
```

通配符：

| 符号 | 说明 |
|------|------|
| `*` | 匹配任意单个符号 |
| `..` | 匹配多个连续任意符号 |
| `+` | 匹配子类类型 |

## 五种通知类型

![通知类型](https://img.zhaizz.top/file/blog-website/spring/1630166147697(1).png)

| 注解 | 说明 |
|------|------|
| `@Before` | 前置通知，方法执行前 |
| `@After` | 后置通知，方法执行后（不管是否异常） |
| `@Around` | **环绕通知（重点）**，方法前后均可增强 |
| `@AfterReturning` | 返回后通知，正常执行完毕后 |
| `@AfterThrowing` | 异常后通知，抛出异常后 |

环绕通知注意事项：
- 必须依赖 `ProceedingJoinPoint` 参数，其他依赖 `JoinPoint` 参数
- 必须调用 `pjp.proceed()` 执行原始方法
- 返回值建议设为 `Object`
- 必须处理 `Throwable` 异常

## 获取数据

```java
// 获取参数
Object[] args = jp.getArgs();            // 非环绕通知
Object[] args = pjp.getArgs();           // 环绕通知

// 获取返回值
Object ret = pjp.proceed();              // 环绕通知
@AfterReturning(value="pt()", returning="ret")
public void afterReturning(JoinPoint jp, Object ret){}  // 返回后通知(JoinPoint可选,但必须放在第一位)

// 获取异常
@AfterThrowing(value="pt()", throwing="t")
public void afterThrowing(JoinPoint jp, Throwable t){}  // 异常后通知(JoinPoint可选,但必须放在第一位)
```
