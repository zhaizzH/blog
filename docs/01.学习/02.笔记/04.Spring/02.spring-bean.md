# Spring Bean 配置与依赖注入

## Bean 配置

### 基础配置

```xml
<bean id="" class="" name="" scope="" init-method="" destroy-method=""/>
```

![Bean 基础配置](https://img.zhaizz.top/file/blog-website/spring/image-20210729183500978(1).png)

| 属性 | 说明 |
|------|------|
| id | Bean 的名称，同一配置文件中唯一 |
| class | Bean 的类全名 |
| name | 别名，可用逗号、分号、空格分隔 |
| scope | 作用范围：`singleton`（默认单例）/ `prototype`（非单例） |

### Bean 实例化方式

| 方式 | 说明 |
|------|------|
| 构造方法 | **常用**，Spring 底层使用反射调用无参构造 |
| 静态工厂 | 了解，兼容老系统 |
| 实例工厂 | 了解 |
| FactoryBean | 实用，Spring 整合其他框架时使用 |

### Bean 生命周期

```java
// 配置方式
<bean init-method="init" destroy-method="destroy"/>

// 注解方式
@PostConstruct   // 初始化方法
@PreDestroy      // 销毁方法
```

![Bean 生命周期](https://img.zhaizz.top/file/blog-website/spring/1629791473409.png)

生命周期：创建对象 → 执行构造方法 → 执行属性注入 → 执行初始化方法 → 使用 Bean → 执行销毁方法

关闭容器：
- `ctx.close()`：调用时关闭
- `ctx.registerShutdownHook()`：JVM 退出前关闭（推荐）

## 依赖注入

### Setter 注入

```xml
<!-- 引用类型 -->
<property name="bookDao" ref="bookDao"/>
<!-- 简单类型 -->
<property name="databaseName" value="mysql"/>
```

### 构造器注入

```xml
<constructor-arg name="bookDao" ref="bookDao"/>
<constructor-arg name="databaseName" value="mysql"/>
```

> 自己开发的模块推荐使用 setter 注入

### 自动装配

```xml
<bean id="bookService" class="..." autowire="byType"/> <!-- 按类型（常用） -->
<bean id="bookService" class="..." autowire="byName"/> <!-- 按名称 -->
```

### 集合注入

```xml
<property name="array"><array><value>100</value></array></property>
<property name="list"><list><value>itcast</value></list></property>
<property name="set"><set><value>itcast</value></set></property>
<property name="map"><map><entry key="country" value="china"/></map></property>
<property name="properties"><props><prop key="country">china</prop></props></property>
```

## 第三方 Bean 管理

### XML 方式

```xml
<context:property-placeholder location="jdbc.properties"/>
<bean id="dataSource" class="com.alibaba.druid.pool.DruidDataSource">
    <property name="driverClassName" value="${jdbc.driver}"/>
    <property name="url" value="${jdbc.url}"/>
    <property name="username" value="${jdbc.username}"/>
    <property name="password" value="${jdbc.password}"/>
</bean>
```

### 注解方式（@Bean）

```java
@Configuration
public class JdbcConfig {
    @Bean
    public DataSource dataSource(){
        DruidDataSource ds = new DruidDataSource();
        // 设置属性...
        return ds;
    }
}
```

使用 `@Import` 引入配置类：

```java
@Configuration
@Import({JdbcConfig.class})
public class SpringConfig { }
```

## 核心容器

| 类 | 加载方式 | 说明 |
|------|------|------|
| ClassPathXmlApplicationContext | XML 配置文件 | **常用**，立即加载 |
| FileSystemXmlApplicationContext | 文件系统 XML | 不推荐，耦合度高 |
| AnnotationConfigApplicationContext | 配置类 | 纯注解开发 |

```java
// 三种获取 Bean 的方式
BookDao bookDao = (BookDao) ctx.getBean("bookDao");
BookDao bookDao = ctx.getBean("bookDao", BookDao.class);
BookDao bookDao = ctx.getBean(BookDao.class);
```
