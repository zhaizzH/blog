# Maven 基础

## 下载安装

下载地址：https://maven.apache.org/download.cgi

![下载页面](https://img.zhaizz.top/file/blog-website/maven/Htk4ngxZ.png)

## 环境变量配置

配置 `MAVEN_HOME` 环境变量：

![MAVEN_HOME](https://img.zhaizz.top/file/blog-website/maven/nrUkthrc.png)

在 `Path` 中添加 `%MAVEN_HOME%\bin`：

![Path 配置](https://img.zhaizz.top/file/blog-website/maven/TCRsP7HM.png)

> 前提：需要先配置好 `JAVA_HOME`

## 仓库配置

### 本地仓库

配置仓库默认存储位置，默认位置在 `C:\Users\你的用户名\.m2\repository` 下，可以通过更改 Maven 安装路径下的 `conf\settings.xml` 修改：

![本地仓库配置](https://img.zhaizz.top/file/blog-website/maven/lNMKQBMT.png)

### 远程仓库（阿里云镜像）

可通过坐标导入依赖，常用仓库：https://mvnrepository.com/

配置远程仓库，更改 Maven 安装路径下的 `conf\settings.xml`：

![远程仓库配置](https://img.zhaizz.top/file/blog-website/maven/QikNoLDM.png)

```xml
<mirror>
  <id>aliyunmaven</id>
  <mirrorOf>*</mirrorOf>
  <name>阿里云公共仓库</name>
  <url>https://maven.aliyun.com/repository/public</url>
</mirror>
```

## 常用命令

```bash
mvn compile    # 编译
mvn clean      # 清理
mvn test       # 测试
mvn package    # 打包
mvn install    # 安装到本地仓库
mvn deploy     # 推送到远程仓库
```

## Tomcat 插件

配置完成后，通过以下命令启动：

```bash
mvn tomcat7:run
```

```xml
<build>
    <plugins>
        <plugin>
            <groupId>org.apache.tomcat.maven</groupId>
            <artifactId>tomcat7-maven-plugin</artifactId>
            <version>2.1</version>
            <configuration>
                <port>80</port>
                <path>/</path>
            </configuration>
        </plugin>
    </plugins>
</build>
```

## 标准 pom.xml

通过 `<properties>` 定义版本号，再用 `${...}` 引用，方便统一管理：

```xml
<properties>
    <junit.version>4.13.1</junit.version>
</properties>

<dependencies>
    <dependency>
        <groupId>junit</groupId>
        <artifactId>junit</artifactId>
        <version>${junit.version}</version>
        <scope>test</scope>
    </dependency>
</dependencies>
```

```xml
<?xml version="1.0" encoding="UTF-8"?>
<project xmlns="http://maven.apache.org/POM/4.0.0"
         xmlns:xsi="http://www.w3.org/2001/XMLSchema-instance"
         xsi:schemaLocation="http://maven.apache.org/POM/4.0.0 http://maven.apache.org/xsd/maven-4.0.0.xsd">
    <!-- 指定 pom 模型版本 -->
    <modelVersion>4.0.0</modelVersion>
    <!-- 打包方式，web 工程打包为 war，java 工程打包为 jar -->
    <packaging>war</packaging>
    <!-- 组织 id -->
    <groupId>top.zhaizz</groupId>
    <!-- 项目 id -->
    <artifactId>maven</artifactId>
    <!-- 版本号 -->
    <version>1.0-SNAPSHOT</version>

    <properties>
        <maven.compiler.source>17</maven.compiler.source>
        <maven.compiler.target>17</maven.compiler.target>
        <project.build.sourceEncoding>UTF-8</project.build.sourceEncoding>
    </properties>

    <!-- 当前工程的所有依赖 -->
    <dependencies>
        <!-- 具体依赖 -->
        <dependency>
            <groupId>junit</groupId>
            <artifactId>junit</artifactId>
            <version>4.13.1</version>
            <scope>test</scope>
        </dependency>
    </dependencies>

    <!-- 构建 -->
    <build>
        <plugins>
            <plugin>
                <groupId>org.apache.tomcat.maven</groupId>
                <artifactId>tomcat7-maven-plugin</artifactId>
                <version>2.1</version>
                <configuration>
                    <port>80</port>
                </configuration>
            </plugin>
        </plugins>
    </build>
</project>
```
