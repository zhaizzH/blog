# JetBrains 全家桶安装和激活

## 准备工作

- Windows 电脑
- 已安装 JetBrains IDE (如 [IntelliJ IDEA](https://www.jetbrains.com/zh-cn/idea/download/other/), [PyCharm](https://www.jetbrains.com.cn/pycharm/download/other/) 等)

> 注意需要激活的版本不要超过网站提示的版本

![jetbrains-activate.png](https://img.zhaizz.top/file/blog-website/kvxP0w1L.png)

## 激活步骤

### 1. 访问导航站

打开 [https://3.jetbra.in/](https://3.jetbra.in/)，选择状态为 **Online** 的网址并点击进入

### 2. 下载激活脚本

点击网页上方的 **"Download jetbra.zip"** 下载压缩包

### 3. 解压并运行脚本

1. 解压 `jetbra.zip` 到一个固定的文件夹(**该文件夹不能删除**，删除后激活会失效)
2. 打开 `scripts` 文件夹，双击 `install-all-users.vbs` 或者 `install-current-user.vbs`
3. 等待约 10 秒，弹出 `Done` 提示框即完成

### 4. 复制激活码

回到刚才的网址，找到你要激活的软件，点击对应激活码即可自动复制

### 5. 激活 IDE

1. 打开需要激活的 JetBrains IDE
2. 在激活窗口点击激活按钮，粘贴激活码，点击激活即可

## 注意事项

✅ 解压后的文件夹不可删除，否则激活失效

✅ 必须下载与激活码对应的 IDE 版本

⚠️ 此类激活仅限个人学习使用，请支持正版
