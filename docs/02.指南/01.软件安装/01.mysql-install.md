# MySQL 8.4.9 安装教程（Windows）

## 下载 MySQL

1. 点击[下载链接](https://dev.mysql.com/downloads/mysql/8.0.html)，进入 MySQL 下载页面
2. 选择版本为 **8.4.9 LTS**
3. 选择操作系统为 **Microsoft Windows**
4. 下载 **Windows (x86, 64-bit), MSI Installer** 版本

![mysql-install.png](https://img.zhaizz.top/file/blog-website/1778138623314_mysql-install.png)

::: info 提示
下载页面有两个 ZIP 选项：
- **ZIP Archive** — 完整版，包含所有组件
- **ZIP Archive (debug binaries)** — 调试版，仅用于开发调试，建议下载普通 ZIP Archive 版本
:::

## 安装步骤

## MSI 安装步骤

### 1. 运行 MSI 安装程序

双击下载的 `mysql-xx.msi` 文件，启动 MySQL Installer

### 2. 选择安装类型

在安装向导中，选择 **"Typical"** 安装类型，或者使用 **"Custom"** 这样可以详细控制安装的组件并更改安装位置

### 3. 执行安装

在安装确认页面点击 **"Install"** 执行安装

### 4. 产品配置

#### Server Configuration Type
- **Config Type**: 选择 **"Development Computer"**（开发电脑）或根据需要选择
- **Connectivity**: 通常选择 **"TCP/IP"**，端口默认为 3306
- **Open Windows Firewall ports...**: 建议勾选，以便允许网络访问

#### Accounts and Roles
1. **MySQL Root Password**: 设置 root 用户的密码（请记住这个密码）
2. **Repeat Password**: 再次输入密码确认
3. 可以点击 **"Add User"** 创建其他用户

#### Windows Service
- **Windows Service Name**: 默认为 `MySQL84`
- **Start the MySQL Server at System Startup**: 建议勾选，让 MySQL 随系统启动

#### Apply Configuration
点击 **"Execute"** 应用配置，等待所有步骤完成

### 5. 完成安装

1. 配置完成后显示 "Configuration Complete"
2. 点击 **"Finish"** 完成安装
3. 安装程序会提示是否启动 MySQL Workbench（如果安装了）

### 6. 配置环境变量（可选）

MSI 安装通常会自动添加环境变量如果没有，可以手动添加：
1. 打开系统环境变量设置
2. 在 Path 中添加 MySQL 的 bin 目录，默认路径为：
```
C:\Program Files\MySQL\MySQL Server 8.4\bin
 ```

### 7. 验证安装

安装完成后，MySQL 服务会自动启动可以通过以下方式验证：

1. **命令提示符验证**:
```CMD
mysql --version
```

正常会输出类似:

```CMD
mysql  Ver 8.0.x for Win64 on x86_64 (MySQL Community Server - GPL)
```

2. **登录 MySQL**:
```CMD
mysql -u root -p
```
输入刚才设置的 root 密码,应该能成功登录

### 8. 修改配置文件（可选）

如果需要自定义配置,可以编辑 MySQL 配置文件:
- 配置文件位置：`C:\ProgramData\MySQL\MySQL Server 8.4\my.ini`
- 修改后需要重启 MySQL 服务:
  ```CMD
  net stop MySQL80
  net start MySQL80
  ```

### 9. 设置远程连接（可选）

如果需要允许远程连接，登录 MySQL 后执行：

```SQL
USE mysql;
UPDATE user SET host = '%' WHERE user = 'root' AND host = 'localhost';
FLUSH PRIVILEGES;
```

::: warning 注意
生产环境不建议开启远程连接，尤其不建议将 root 用户设置为远程访问，存在安全风险
:::

## 常用命令

| 命令 | 说明 |
| ---- | ---- |
| `net start mysql` | 启动 MySQL 服务 |
| `net stop mysql` | 停止 MySQL 服务 |
| `mysqld --remove` | 卸载 MySQL 服务 |
| `mysql -u root -p` | 登录 MySQL |

## 注意事项

✅ 初始化时务必使用**管理员身份**运行终端

✅ 初始密码仅在控制台输出一次,**务必保存**

✅ 如果 3306 端口被占用，可以在 `my.ini` 中修改 `port` 配置

⚠️ 卸载 MySQL 前请先备份 data 目录中的数据

⚠️ 如果安装失败，检查是否已安装 VC++ 运行库（MySQL 8.0 依赖 Microsoft Visual C++ Redistributable）
