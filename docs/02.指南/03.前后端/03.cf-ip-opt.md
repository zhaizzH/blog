# 使用 Cloudflare 配置优选 IP

:::tip
所有优选一个域名即可,无需两个域名,如: a.yourdomain.com 和 b.yourdomain.com 即可完成优选
:::

众所周知,Cloudflare 号称免费提供域名解析,CDN 等功能,但 Cloudflare 的 CDN 对于中国大陆的用户来说,真是一言难尽,所以根据博主[二叉树树](https://2x.nz/)的[教程](https://2x.nz/posts/cf-fastip/)写下了这篇教程

## 选择优选域名

优选的核心就是选择一个国内访问速度更快的Cloudflare节点IP或域名,常用的社区优选域名:https://cf.090227.xyz , https://cf.877774.xyz/ ,这些优选域名通常是通过扫描Cloudflare官方IP段,找出国内延迟最低的IP整理而成

## 准备工作

- 一个托管在 Cloudflare 的域名（如 `yourdomain.com`）
- 准备一个 **未使用** 的子域名用于回源,例如 `origin.yourdomain.com`

## 各类优选方法

### Saas 优选

:::tip
需要将 SSL/TLS →概述→ SSL/TLS 加密改为灵活
:::

```
用户 → xxxx.yourdomain.com （CNAME 到优选域名 → 低延迟 Cloudflare IP）
                    ↓
          Cloudflare 边缘节点（根据 SaaS 规则回源）
                    ↓
          origin.yourdomain.com （CNAME 到你的网站）
                    ↓
                 返回网站
```

#### 创建回源记录

进入 Cloudflare 仪表板 → 你的域名 → **DNS** → 添加 **CNAME** 记录:

![{40F476EE-93AE-448B-AC47-62B54C6C03D5}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/hltapBzF.png)

:::tip
或者添加 **A** 记录

![{8FED0F85-D407-4BD1-B1A8-8A434E555BD9}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/7Y0I0SZx.png)

:::

添加一个回退源,名字和指向随便写

![{27F3CEED-B82F-4A86-BA82-498425BD71AC}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/91WbudQK.png)

#### 启用 Cloudflare for SaaS

1. 在域名菜单中选择 **SSL/TLS** → **自定义主机名**
2. 首次使用需开通 Cloudflare for SaaS（需绑定支付方式,个人额度免费,不产生费用）
3. 开通后,在 **回退源** 中填入 `saas.zhaizz.top`,等待状态变为 **有效**

![{36ADC97A-6264-46EE-BEF6-DD68444D4114}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/43O84Qo2.png)

4. 点击 **添加自定义主机名**,输入你的站点域名,如 `blog.zhaizz.top`
5. 证书验证方法使用 **HTTP** 验证
6. 自定义源服务器使用先前的 `origin.zhaizz.top`,然后添加自定义主机名

![{27A67E41-FC36-4AD7-8CD2-13B56F4D7D13}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/nhQq7goR.png)

7. 等待验证通过,自定义主机名状态显示为 **有效**

![{D624C4AF-2369-461F-BC00-A6202BEA9553}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/aKbFSIFE.png)

#### 配置域名

在你的域名添加 CNAME 解析指向优选节点,如 `优选.cf.877774.xyz`,不开启 cf 代理

![{B064EE71-3A2D-4082-BC52-9BB1CDB32D4B}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/Pb7C4yqm.png)

最后在你的域名添加 CNAME 解析,名称为 `blog`,目标为刚才的 `cdn.zhaizz.top`,不开启 cf 代理

![{0CD69409-7609-45AC-B466-837738D09E1C}.png](https://img.zhaizz.top/file/blog-website/cf-ip-opt/Ihh6QkXR.png)
