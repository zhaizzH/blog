# 随身 WIFI 刷 Debian 系统并部署青龙面板

## 前言

本文适用于使用 **高通 410 芯片 (JZ02_V10)** 的随身 WIFI，全程覆盖刷 Debian 系统 → 系统初始化 → 安装 Docker → 部署青龙面板

![IMG_20260522_180353_edit_33503406949054.jpg](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/bN8SCadr.jpg)

根据酷安 [非一般的焰火](http://www.coolapk.com/u/1924236) 的 [教程](https://www.coolapk.com/feed/67348576)编写

## 准备工作

### 硬件清单

- 高通 410 板子 (JZ02_V10)
- Windows 电脑

### 软件清单

- [Debian 系统恢复包](https://pan.quark.cn/s/346b2a672707)
- [9008 驱动](https://pan.quark.cn/s/45a585398f78)
- [Miko 工具](https://pan.quark.cn/s/e322377715d1)
- [SSH 远程工具](https://pan.quark.cn/s/9498c64d8afe)
- [原版安卓系统恢复包 (可选)](https://pan.quark.cn/s/add95590fd9e)

::: info 提示
以上资源均来自于互联网,已有资源的可跳过下载
:::

## 刷入 Debian 系统

### 1. 安装 Miko 工具

解压 `Miko_Loder Pro.zip`，运行 `miko.exe` 安装备份工具安装完成后，将解压得到的 `Loader.exe` 复制到 Miko 安装目录，打开 `Loader.exe` 即可

同时安装 9008 驱动

### 2. 进入 9008 刷机模式

1. 拆机找到板子上的按钮,按住的同时插入电脑 USB 口

![IMG_20260522_180121_edit_33340295976683.jpg](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/v62ITuXj.jpg)

2. 打开设备管理器确认出现 **9008** 端口(此时板子不亮灯)
3. 打开 Miko 工具，选择 Debian 系统恢复包刷入

![{1F96992D-E125-4969-8843-C81F397F3289}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/K9KmdWmW.png)

4. 等待进度条到 100%,板子会自动重启亮灯,此时拔掉 USB 再重新插上

### 3. 安装 RNDIS 网络驱动

板子重新连接后,设备管理器中会出现一个未知设备：

![{8FF40B68-E72C-45AE-86E1-BF99C12F52F6}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/CgF8tZqa.png)

1. 右键该设备 → **更新驱动程序** → **浏览我的电脑以查找驱动程序**

![{D5ED5C01-3E00-4123-A16F-BD93E389BFF5}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/rVB7AauH.png)

2. 选择 **从计算机上的可用驱动程序列表中选取** → **显示所有设备**
3. 左侧选择 **Microsoft**,右侧选择 **基于远程 NDIS 的 Internet 共享设备**

![{84256A0F-C0AC-4537-BF94-263279EA8CF7}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/tieKOdgj.png)

4. 在弹出的窗口中确认

至此 Debian 系统已成功刷入

## 初始化系统

### 1. SSH 连接

打开 SSH 远程工具连接板子,默认密码为 `1313144`,连接成功后建议立即修改密码

![{C22E0C81-57D5-49D7-A9C9-4DD7DEED8A84}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/1V9TM7Xm.png)

### 2. 连接 WiFi

执行 `nmtui` 进入网络配置界面：

```bash
nmtui
```

选择 **启用连接**，找到自己的 WiFi 并连接，即可联网

![{A99D78A6-71F3-4BB0-8648-675F0E766B96}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/eck1sGrD.png)

## 安装 Docker

### 1. 更新软件包

```bash
sudo apt-get update
```

### 2. 安装 curl 和 wget

```bash
apt install curl wget
```

### 3. 安装 Docker

```bash
apt install docker.io
```

### 4. 设置 Docker 自启动

```bash
systemctl enable docker
systemctl start docker
```

### 5. 配置镜像加速

编辑 Docker 配置文件：

```bash
sudo nano /etc/docker/daemon.json
```

粘贴以下内容：

```json
{
  "registry-mirrors": [
    "https://docker.m.daocloud.io",
    "https://docker.imgdb.de",
    "https://docker-0.unsee.tech",
    "https://docker.hlmirror.com",
    "https://docker.1ms.run",
    "https://func.ink",
    "https://lispy.org",
    "https://docker.xiaogenban1993.com"
  ]
}
```

::: info 提示
若 `daemon.json` 文件不存在,`nano` 会自动创建该文件
:::

重启 Docker 使加速生效：

```bash
sudo systemctl daemon-reload
sudo systemctl restart docker
```

## 部署青龙面板

执行以下命令以 Docker 方式部署青龙面板：

```bash
docker run -dit \
  -v $PWD/qinglong:/ql/data \
  -p 5700:5700 \
  --name qinglong \
  --hostname qinglong \
  --restart unless-stopped \
  whyour/qinglong:latest
```

部署完成后,浏览器访问 `ip:5700` 进入青龙面板后台

![{5211CDC4-476A-4933-8DA4-0DFFF99D833C}.png](https://img.zhaizz.top/file/blog-website/portable-wifi-flash/EVcE9Pcl.png)

## 注意事项

✅ 刷机前确保 9008 驱动已正确安装

✅ 刷机完成后第一时间修改 SSH 默认密码

✅ 安装 Docker 后务必验证 `docker --version` 确认安装成功

✅ 青龙面板部署后检查容器运行状态：`docker ps`

⚠️ 嵌入式设备资源有限,不建议同时运行过多脚本

⚠️ 若安装失败,检查网络连接是否正常,镜像加速是否配置正确
