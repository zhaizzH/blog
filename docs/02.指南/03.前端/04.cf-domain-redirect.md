# Cloudflare 根域名重定向到 WWW 域名

参考 [jopny](https://www.cnblogs.com/jopny/) 的[教程](https://www.cnblogs.com/jopny/p/18798822/cloudflare_dns_redirect_2_www_rule)

将托管在 Cloudflare 的根域名（如 `zhaizz.top`）301 永久重定向到 `www.zhaizz.top`，避免搜索引擎收录重复内容

## 1. 配置根域名 DNS 记录

进入 Cloudflare 仪表盘 → 你的域名 → **DNS** → 添加记录:

| 字段 | 值 |
|------|-----|
| 类型 | **CNAME** |
| 名称 | **@**（代表根域名） |
| 目标 | **www.zhaizz.top**（你的 www 域名） |
| 代理状态 | **开启**（橙色云朵） |

> 代理状态必须开启，否则重定向规则不会生效

## 2. 使用模板创建重定向规则

1. 左侧导航栏点击 **规则** → **模板**
2. 选择 **从根重定向到WWW**
3. 使用默认设置，直接点击 **保存**

![cf-domain-redirect.png](https://img.zhaizz.top/file/blog-website/RTXjoBU4.png)

Cloudflare 会自动创建一条重定向规则，将根域名的所有请求 301 重定向到对应的 `www` 域名

## 3. 验证重定向是否生效

浏览器访问 `https://zhaizz.top`，地址栏会自动跳转到 `https://www.zhaizz.top`

本地测试命令:

```bash
curl -I https://zhaizz.top
```

返回 `301 Moved Permanently`，且 `Location` 指向 `https://www.zhaizz.top/` 即代表配置成功

## 补充说明

### 使用页面规则实现

也可以使用 **页面规则** 实现相同效果:

1. 左侧导航栏点击 **规则** → **页面规则**
2. 点击 **创建页面规则**
3. URL 匹配: `zhaizz.top/*`
4. 添加设置: **转发 URL** → **状态码: 301** → **目标 URL: `https://www.zhaizz.top/$1`**

### 使用重定向规则实现

或者使用 **重定向规则（Redirect Rules）**:

1. 左侧导航栏点击 **规则** → **重定向规则**
2. 点击 **创建重定向规则**
3. 规则名称: `根域名重定向到 www`
4. 当请求匹配: `主机名 等于 zhaizz.top`
5. 重定向 URL: `https://www.zhaizz.top`
6. 状态码: `301`

### DNS 记录替代方案

如果不想使用 CNAME 记录，也可以添加一条 **A 记录** 指向占位 IP `192.0.2.1`，同样需要开启代理

## 注意事项

✅ 必须开启代理（橙色云朵），否则规则不生效

✅ 确保 `www.zhaizz.top` 的 A/CNAME 记录已正确配置

✅ 等待 DNS 缓存刷新（通常 1～10 分钟），重定向即可全局生效
