# 制作 LinuxToGo

## 前言

LinuxToGo 是将完整 Linux 系统安装到 USB 移动硬盘或 U 盘上,使其可以随身携带、即插即用的技术。

本文使用 **Ventoy** 工具制作 LinuxToGo。

## 准备工作

### 硬件要求

| 项目 | 最低要求 | 推荐配置 |
| ---- | -------- | -------- |
| 移动存储设备 | USB 3.0 U 盘（16GB+） | USB 3.0/3.1 移动固态硬盘（64GB+） |
| 读取速度 | 30MB/s | 100MB/s 以上 |
| 写入速度 | 10MB/s | 40MB/s 以上 |
| 接口类型 | USB 3.0 | USB 3.1/3.2 |

::: warning 注意
强烈推荐使用 **移动固态硬盘（SSD）** 作为 LinuxToGo 盘。普通 U 盘虽然也能运行,但读写速度较慢,会影响系统体验。
:::

### 软件准备

- Linux Mint ISO 镜像
- Ventoy 启动盘制作工具
- 一台正常运行的电脑

### 下载 Linux Mint ISO

从 Linux Mint 官网下载最新稳定版:

- [Linux Mint 官网下载](https://linuxmint.com/download.php)

Linux Mint 提供三个桌面环境版本:

| 版本 | 桌面环境 | 适合人群 |
| ---- | -------- | -------- |
| **Cinnamon** | Cinnamon（推荐） | 功能最全,适合大多数用户 |
| **MATE** | MATE | 老旧硬件,资源占用更低 |
| **Xfce** | Xfce | 极低配置电脑 |

::: info 提示
推荐下载 **Cinnamon 版本**,它是 Linux Mint 的默认桌面环境,功能最完整。如果电脑配置较低,可以选择 MATE 或 Xfce 版本。
:::

### 下载 Ventoy

从 Ventoy 官网下载最新版本:

- [Ventoy 官网下载](https://www.ventoy.net/cn/download.html)

下载对应系统平台的安装包:
- **Windows**：`ventoy-x.x.xx-windows.zip`
- **Linux**：`ventoy-x.x.xx-linux.tar.gz`
- **macOS**：`ventoy-x.x.xx-mac.zip`

## 制作步骤

### 1. 连接移动硬盘

将移动固态硬盘通过 USB 3.0 或更高版本接口连接到电脑

::: warning 注意
请确保移动硬盘上没有重要数据,Ventoy 安装过程会 **格式化** 整个磁盘。
:::

### 2. 安装 Ventoy 到移动硬盘

解压下载的 Ventoy 压缩包,进入解压后的目录:

**Windows 用户：**

双击运行 `Ventoy2Disk.exe`

### 3. 选择目标设备

在 Ventoy 界面的 **"设备"** 下拉菜单中选择你的移动硬盘

::: danger 警告
请仔细确认设备名称和容量,**选错磁盘会导致该磁盘所有数据丢失**。建议拔掉其他 USB 存储设备,只保留目标磁盘。
:::

### 4. 分区类型选择

在安装前,根据目标电脑选择分区类型:

| 目标电脑类型 | 推荐选择 |
| ------------ | -------- |
| 较新电脑（UEFI） | **GPT** |
| 较旧电脑（Legacy BIOS） | **MBR** |
| 需要兼容两种 | **MBR**（兼容性更好） |

::: info 提示
Ventoy 默认使用 GPT 分区表。如果需要兼容老旧电脑的 BIOS 启动,可以在安装前点击 **"配置选项"** 切换为 MBR。
:::

### 5. 安装 Ventoy

确认设置无误后,点击 **"安装"** 按钮。Ventoy 会弹出两次确认提示,确认后开始安装

安装过程非常快,通常几秒钟即可完成

### 6. 复制 Linux Mint ISO 到移动硬盘

安装完成后,U 盘会被格式化为一个大容量的 EXFAT 分区。将下载好的 Linux Mint ISO 文件直接复制到这个分区的根目录即可

```
USB 移动硬盘（Ventoy 分区）/
├── linuxmint-22.1-cinnamon-64bit.iso
```

::: info 提示
这正是 Ventoy 的优势所在：无需使用任何写入工具,像复制普通文件一样将 ISO 拖入 U 盘即可。如果以后想添加其他系统,只需再复制对应的 ISO 文件即可。
:::

### 7. 启用持久化存储（可选）

如果希望在 LinuxToGo 中保存文件和安装的软件,需要创建持久化存储文件:

1. 在 Ventoy 分区根目录创建一个名为 `ventoy/dat disk.dat` 的文件（注意路径中的空格）
2. 该文件即为持久化存储空间,文件大小决定了可用的持久化空间

使用 Windows 创建固定大小文件:

```cmd
:: 打开管理员 CMD,创建一个 30GB 的持久化存储文件
fsutil file createnew "E:\ventoy\dat disk.dat" 32212254720
```

使用 Linux 创建:

```bash
# 创建一个 30GB 的持久化存储文件（将 /mnt/ventoy 替换为你的 Ventoy 分区挂载点）
dd if=/dev/zero of="/mnt/ventoy/ventoy/dat disk.dat" bs=1M count=30720 status=progress
```

然后在 Ventoy 配置文件中启用持久化:

```json
// 编辑 Ventoy 分区上的 ventoy/ventoy.json 文件（如不存在则创建）
{
    "persistence": [
        {
            "image": "/linuxmint-22.1-cinnamon-64bit.iso",
            "backend": "/ventoy/dat disk.dat"
        }
    ]
}
```

::: warning 注意
如果不配置持久化存储,LinuxToGo 每次重启后所有数据和更改都会丢失,相当于 Live 模式。
:::

## 首次启动配置

### 1. 设置 BIOS 启动项

将 LinuxToGo 盘插入需要使用的电脑,重启并进入 BIOS:

- **常见 BIOS 进入方式**:
  - 按 `F2`、`F12`、`Del` 或 `Esc` 键（具体按键因主板而异）
  - 或通过系统设置进入高级启动选项

### 2. 调整启动顺序

在 BIOS 中将 USB 设备设置为第一启动项:
- **UEFI 模式**:选择带 "UEFI" 前缀的 USB 设备
- **Legacy 模式**:选择不带 UEFI 前缀的 USB 设备

::: info 提示
部分电脑支持开机时按 `F12`（或其他按键）直接调出启动菜单,无需进入 BIOS 修改启动顺序。
:::

### 3. 进入 Ventoy 启动菜单

启动后会看到 Ventoy 的引导菜单,显示 U 盘中所有可用的 ISO 文件:

| 选项 | 说明 |
| ---- | ---- |
| **linuxmint-22.1-cinnamon-64bit.iso** | 选择此项启动 Linux Mint |
| **memtest86+** | 内存检测工具（Ventoy 内置） |

选择 Linux Mint ISO 后按回车,进入 Linux Mint 启动菜单

### 4. 选择启动模式

进入 Linux Mint 后会看到 GRUB 引导菜单:

| 选项 | 说明 |
| ---- | ---- |
| **Start Linux Mint** | 直接进入 Linux Mint 桌面（推荐） |
| **Start Linux Mint (compatibility mode)** | 兼容模式,硬件有问题时使用 |
| **OEM install (for manufacturers)** | OEM 安装模式 |
| **Check the integrity of the medium** | 检查 ISO 完整性 |
| **Boot from local hard drive** | 从本地硬盘启动 |

选择 **"Start Linux Mint"** 进入系统

### 5. 系统初始化

首次进入系统后,Linux Mint 已经配置好了中文环境。你需要:

1. **连接 Wi-Fi**：点击右下角网络图标,选择可用的无线网络并输入密码
2. **检查系统**：确认桌面环境正常显示,网络连接正常

### 6. 安装到移动硬盘（完整安装）

如果不想使用 Live 模式,想将系统完整安装到移动硬盘:

1. 在桌面双击 **"Install Linux Mint"** 图标
2. 选择语言为 **"中文（简体）"**
3. 键盘布局选择 **"English (US)"**
4. 勾选 **"安装多媒体编解码器"**（推荐）
5. 安装类型选择 **"清除磁盘并安装 Linux Mint"**
6. **重要**：在选择磁盘时,务必选择你的 USB 移动硬盘,不要选错到本地硬盘
7. 设置时区、用户名和密码
8. 等待安装完成,重启后拔掉本地硬盘的启动介质即可

::: danger 警告
安装过程中选择磁盘时请 **再三确认**,选错磁盘会导致本地系统被覆盖,造成不可挽回的数据损失。
:::

::: info 提示
完整安装到移动硬盘后,Linux Mint 的使用体验与本地安装完全一致,所有更改和文件都会被保留。
:::

### 7. 常用软件安装

进入系统后,建议安装以下常用软件:

```bash
# 更新软件源
sudo apt update && sudo apt upgrade -y

# 安装常用工具
sudo apt install -y wget curl git vim net-tools

# 安装输入法（Fcitx5 + 拼音）
sudo apt install -y fcitx5 fcitx5-chinese-addons fcitx5-config-qt

# 安装 Chrome 浏览器
wget -q -O - https://dl.google.com/linux/linux_signing_key.pub | sudo gpg --dearmor -o /usr/share/keyrings/google-chrome.gpg
echo "deb [arch=amd64 signed-by=/usr/share/keyrings/google-chrome.gpg] http://dl.google.com/linux/chrome/deb/ stable main" | sudo tee /etc/apt/sources.list.d/google-chrome.list
sudo apt update && sudo apt install -y google-chrome-stable

# 安装常用多媒体播放器
sudo apt install -y vlc ffmpeg
```

### 8. 配置中文输入法

Linux Mint 默认使用 IBus 输入法框架,需要安装中文输入法:

```bash
# 安装中文输入法引擎
sudo apt install -y ibus-libpinyin ibus-table-wubi

# 设置输入法
im-config -n ibus
```

设置完成后重启系统,在系统托盘中点击键盘图标切换输入法,或使用 `Super + Space` 快捷键切换

## 进阶配置

### 配置固定 IP（可选）

如果你需要在不同网络环境中使用固定的 IP 地址:

```bash
# 编辑 netplan 配置文件
sudo nano /etc/netplan/01-netcfg.yaml
```

```yaml
network:
  version: 2
  renderer: NetworkManager
  ethernets:
    enx0:  # 替换为你的网卡名称（通过 ip addr 查看）
      dhcp4: no
      addresses:
        - 192.168.1.100/24
      routes:
        - to: default
          via: 192.168.1.1
      nameservers:
        addresses: [223.5.5.5, 8.8.8.8]
```

```bash
# 应用配置
sudo netplan apply
```

### 多系统启动

Ventoy 最大的优势是支持多系统启动。你可以在同一个 USB 盘中存放多个 ISO,按需选择启动:

```
USB 移动硬盘（Ventoy 分区）/
├── linuxmint-22.1-cinnamon-64bit.iso
├── ubuntu-24.04-desktop-amd64.iso
├── fedora-workstation-40.iso
├── ventoy/
│   └── ventoy.json
```

启动时 Ventoy 会列出所有可用的 ISO,选择对应系统即可进入

### 优化 SSD 性能

如果使用的是移动固态硬盘,建议进行以下优化:

```bash
# 查看 TRIM 是否已启用
sudo fstrim -av

# 启用定期 TRIM（通常默认已启用）
sudo systemctl enable fstrim.timer
```

## 注意事项

✅ 制作前务必备份移动硬盘上的重要数据

✅ 推荐使用 USB 3.0 及以上接口的移动固态硬盘

✅ 不同电脑的 BIOS 设置可能不同,启动前请确认 BIOS 已启用 USB 启动

✅ Ventoy 支持同时存放多个 ISO,可以在一台 U 盘上制作多个系统的 LinuxToGo

⚠️ 不要在 LinuxToGo 系统运行时直接拔出移动硬盘,可能导致数据损坏或文件系统错误

⚠️ Live 模式下所有更改重启后都会丢失,如需保留数据请配置持久化存储或完整安装

⚠️ 部分电脑的安全启动（Secure Boot）可能会阻止 Ventoy 启动,如遇问题请在 BIOS 中暂时关闭 Secure Boot

⚠️ Linux Mint 在不同电脑上可能遇到显卡驱动兼容问题,可尝试使用 compatibility mode 模式启动
