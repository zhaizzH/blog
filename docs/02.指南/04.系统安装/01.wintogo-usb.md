# 制作 WinToGo 移动硬盘

## 前言

WinToGo（Windows To Go）是将完整 Windows 系统安装到 USB 移动硬盘或 U 盘上,使其可以随身携带、即插即用的技术。适用于以下场景：

- 需要在不同电脑上使用统一的 Windows 环境
- 不想在主硬盘上安装双系统
- 临时需要一台可用的 Windows 电脑
- 需要便携式工作环境

::: info 提示
本文使用 [萝卜头论坛](https://bbs.luobotou.org/bstra/thread-761-1-1.html) 的工具和方法编写
:::

## 准备工作

### 硬件要求

| 项目 | 最低要求 | 推荐配置 |
| ---- | -------- | -------- |
| 移动存储设备 | USB 3.0 U 盘（32GB+） | USB 3.0/3.1 移动固态硬盘（128GB+） |
| 读取速度 | 50MB/s | 150MB/s 以上 |
| 写入速度 | 20MB/s | 50MB/s 以上 |
| 接口类型 | USB 3.0 | USB 3.1/3.2 |

::: warning 注意
不建议使用普通机械硬盘或低速 U 盘,会导致系统运行极其卡顿。强烈推荐使用 **移动固态硬盘（SSD）** 作为 WinToGo 盘。
:::

### 软件准备

- Windows 10/11 系统 ISO 镜像
- WinToGo 制作工具（见下方工具介绍）
- 一台正常运行的 Windows 电脑

### 下载 Windows ISO

从微软官网下载对应版本的系统镜像：

- [Windows 11 下载](https://www.microsoft.com/zh-cn/software-download/windows11)
- [Windows 10 下载](https://www.microsoft.com/zh-cn/software-download/windows10ISO)

::: info 提示
推荐使用 **Windows 10/11 专业版或企业版**。家庭版虽然也可以制作 WinToGo,但部分功能可能受限。
:::

本文使用[吻妻 Win10](https://iwin11.cn/page/win10)

## 制作步骤

### 1. 下载系统镜像

从微软官网下载 Windows 10/11 ISO 镜像文件,保存到本地磁盘

### 2. 连接移动硬盘

将移动固态硬盘通过 USB 3.0 或更高版本接口连接到电脑

::: warning 注意
请确保移动硬盘上没有重要数据,制作过程会 **格式化** 整个磁盘。
:::

### 3. 运行 WTG 辅助工具

双击运行 WTG 辅助工具,如果弹出 UAC 提示,点击 **"是"** 允许运行

### 4. 选择 ISO 镜像

点击 **"浏览"** 按钮,选择下载好的 Windows ISO 镜像文件

### 5. 选择目标磁盘

在磁盘列表中选择你的移动硬盘。请仔细确认盘符和容量,避免选错磁盘导致数据丢失

### 6. 选择系统版本

从下拉列表中选择要安装的 Windows 版本,根据镜像中存在的版本选择

### 7. 高级选项（可选）


### 8. 开始写入

确认所有设置无误后,点击 **"部署"** 按钮。制作具体时间取决于硬盘速度和镜像大小

::: info 提示
制作过程中请勿拔出移动硬盘或关闭程序,否则可能导致制作失败或数据损坏。
:::

### 9. 完成制作

写入完成后,工具会提示制作成功。此时你的移动硬盘已经是一个可启动的 WinToGo 系统盘

## 首次启动配置

### 1. 设置 BIOS 启动项

将 WinToGo 盘插入需要使用的电脑,重启并进入 BIOS：

- **常见 BIOS 进入方式**：
  - 按 `F2`、`F12`、`Del` 或 `Esc` 键（具体按键因主板而异）
  - 或通过 Windows 设置 → 更新和恢复 → 高级启动 → 重启

### 2. 调整启动顺序

在 BIOS 中将 USB 设备设置为第一启动项：
- **UEFI 模式**：选择带 "UEFI" 前缀的 USB 设备
- **Legacy 模式**：选择不带 UEFI 前缀的 USB 设备

### 3. 进入 Windows 设置

首次启动会进入 Windows 初始化设置界面：
1. 选择区域和语言
2. 设置用户名和密码
3. 设置隐私选项
4. 等待系统完成配置

### 4. 安装驱动

进入系统后,建议安装必要的驱动程序,建议使用吾爱破解论坛上的[360驱动大师v2.0.0.2050绿色纯净版](https://www.52pojie.cn/thread-1969739-1-1.html)


## 注意事项

✅ 制作前务必备份移动硬盘上的重要数据

✅ 推荐使用 USB 3.0 及以上接口的移动固态硬盘

✅ 首次使用建议安装完整的硬件驱动

✅ 非必要不要更新系统

⚠️ 不要在 WinToGo 系统运行时直接拔出移动硬盘,可能导致数据损坏

⚠️ 不同电脑的 BIOS 设置可能不同,启动前请确认 BIOS 已启用 USB 启动

⚠️ WinToGo 系统不支持 Windows 更新中的大版本升级,建议全新制作

⚠️ 部分安全软件可能将 WinToGo 盘识别为异常设备,属于正常现象
